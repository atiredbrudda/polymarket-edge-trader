Rewire signal detection from ExpertiseScore (raw_score > 70) to LiftScore (quintile == 5) for expert identification. Add price-context enrichment to consensus signals (expert avg entry price). Rewrite the `analyze` CLI command as the primary Q5 identification and signal surface with `--signals` mode showing the real-time decision tree.

Price context (expert avg entry vs market price) is the core actionable insight — "can I enter at or below the Q5 entry?" Analyze becomes the primary command for viewing who the Q5 traders are and what they're doing.

Output: Working signal detection using Q5 experts, `polymarket analyze` showing Q5 leaderboard, `polymarket analyze --signals` showing active Q5 consensus with price context.

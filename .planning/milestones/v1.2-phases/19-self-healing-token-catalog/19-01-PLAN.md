Module structure:

```python
"""Catalog patcher: detects and patches token_catalog gaps after backfill.

Runs automatically at the end of every backfill command. Also callable
standalone via `polymarket patch-catalog`.

3-tier lookup:
  Tier 1: Local join — markets.tokens -> gamma_events.clob_token_ids -> tags
  Tier 2: Gamma API /markets?conditionId= -> tags (batch size 20)
  Tier 3: Fallback — insert with category from markets.category, node_path=NULL
"""
```

Key implementation rules:

DETECTION QUERY — exact SQL (locked):
```python
missing = session.execute(text("""
    SELECT DISTINCT t.market_id
    FROM trades t
    LEFT JOIN token_catalog tc ON t.market_id = tc.condition_id
    WHERE tc.condition_id IS NULL
""")).scalars().all()
```

GAMMA EVENT INDEX — build once per patch call for Tier 1 performance:
```python
# Build {token_id: GammaEvent} index from all gamma_events
# JSON parse event.clob_token_ids for each event
# O(1) lookup per token instead of O(n*m) scan
```

TIER 1 — local lookup:
- Parse market.tokens JSON: `json.loads(market.tokens)` → list of `{"token_id": "...", ...}`
- For each token_id, check gamma_event_index
- If found: call `_extract_classification(json.loads(event.tags))` → (node_path, depth)
- Build row with niche_slug="esports" if node_path starts with "esports", else derive from category
- If market.tokens is None OR no token found in index → add to needs_api list

TIER 2 — Gamma API batch:
- BATCH_SIZE = 20
- URL: `https://gamma-api.polymarket.com/markets`
- params: `[("conditionId", cid) for cid in batch]`
- timeout=10, sleep 0.05 between batches
- Per response market: extract conditionId, clobTokenIds (parse JSON if str), tags, question
- Call `_extract_classification(tags)` → (node_path, depth)
- niche_slug = derive from tags or market.category.lower()
- Build 2 rows (one per token_id in clobTokenIds)
- If clobTokenIds empty or API fails → add to fallback list

TIER 3 — category-only fallback:
- For each condition_id that failed Tier 1 and Tier 2
- Look up market from DB (already fetched)
- Insert with: niche_slug=market.category.lower() (or "unknown"), node_path=None, depth=None
- token_id: skip if we have no token_id at all (no rows to insert — log at DEBUG)
- If market.tokens is not None, use first token_id from JSON as placeholder (we know it's a real token)
- If market.tokens is None AND API had no clobTokenIds: skip (can't insert without token_id)

NICHE_SLUG DERIVATION helper:
```python
def _derive_niche_slug(category: str | None, node_path: str | None) -> str:
    if node_path and node_path.startswith("esports"):
        return "esports"
    if category:
        return category.lower().strip() or "unknown"
    return "unknown"
```

    1. Create a shared setup helper in commands.py (or a separate src/cli/deps.py):
       ```python
       def _get_dependencies(settings=None):
           """Create all pipeline dependencies from settings."""
           from sqlalchemy import create_engine
           from src.db.models import Base

           settings = settings or get_settings()

           # Create engine and session factory
           engine = create_engine(settings.database_url)
           Base.metadata.create_all(engine)  # Auto-create tables if needed
           session_factory = get_session_factory(settings.database_url)

           client = PolymarketClient(
               host=settings.polymarket_api_host,
               max_requests_per_second=settings.max_requests_per_second,
           )

           # Create category filter (taxonomy loaded internally by pipeline if needed)
           category_filter = CategoryFilter(settings.detail_categories)

           # Optional Telegram alerter
           alerter = TelegramAlerter.from_settings(settings)

           return session_factory, client, category_filter, alerter
       ```

    2. Update each CLI command to use _get_dependencies():
       - markets: open session → get_active_markets → format → print
       - trader: open session → query functions → format → print
       - signals: open session → get_ranked_signals → format → print
       - leaderboard: open session → get_game_leaderboard → format → print
       - sweep: get deps → run_sweep → format_sweep_summary → print
       - poll: get deps → run_polling_loop (blocks until SIGINT)

    3. Handle missing Telegram credentials gracefully:
       - In sweep/poll: if alerter is None, log info "Telegram not configured, alerts disabled"
       - In markets/trader/signals/leaderboard: no Telegram needed, no warning

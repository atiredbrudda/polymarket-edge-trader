<feature>
  <name>Sweep Orchestration and Polling Loop</name>
  <files>src/cli/scheduler.py, src/cli/commands.py, src/config/settings.py, tests/test_scheduler.py</files>
  <behavior>
    run_sweep function:
    ```python
    def run_sweep(
        session_factory: sessionmaker,
        client: PolymarketClient,
        category_filter: CategoryFilter,
        alerter: TelegramAlerter | None = None,
        skip_alerts: bool = False,
    ) -> dict
    ```

    Pipeline stages (in order):
    1. Create IngestionPipeline, call run_full_sweep() → capture ingest stats
    2. Open session, call compute_all_game_scores() → capture leaderboard counts
    3. Call refresh_all_signals() → capture signal count
    4. If alerter and not skip_alerts: call deliver_signal_alerts() → capture alert results
    5. Return stats dict: {markets_ingested, traders_discovered, scores_computed, signals_detected, alerts_sent, alerts_failed, duration_seconds}

    Each stage wrapped in try/except — failure in one stage logs error and continues to next.

    run_polling_loop function:
    ```python
    def run_polling_loop(
        session_factory: sessionmaker,
        client: PolymarketClient,
        category_filter: CategoryFilter,
        alerter: TelegramAlerter | None = None,
        interval_minutes: int = 60,
    ) -> None
    ```

    Loop behavior:
    1. Register signal handler for SIGINT and SIGTERM → set shutdown flag
    2. While not shutdown:
       a. Call run_sweep()
       b. Log dense one-line summary:
          `[Timestamp] Cycle N: {markets} mkts, {signals} sigs, {alerts} alerts ({duration}s)`
       c. If not shutdown: sleep for interval_minutes (check shutdown flag periodically)
    3. Log "Polling stopped gracefully"

    Graceful sleep: break sleep into 1-second intervals, checking shutdown flag each iteration.

    poll CLI command (added to commands.py):
    @cli.command()
    @click.option('--interval', default=None, type=int, help='Polling interval in minutes')
    @click.option('--no-alerts', is_flag=True, help='Skip alert delivery')
    def poll(interval, no_alerts): ...

    Uses interval from flag if provided, else from Settings.poll_interval_minutes.

    Settings update:
    - Add poll_interval_minutes: int = 60 to Settings class

    Cases (test_scheduler.py):
    - run_sweep returns stats dict with all expected keys
    - run_sweep continues on ingest failure (logs error, proceeds to scoring)
    - run_sweep continues on scoring failure
    - run_sweep skips alerts when skip_alerts=True
    - run_sweep skips alerts when alerter is None
    - run_polling_loop respects shutdown flag (mock signal)
    - run_polling_loop logs cycle stats
    - All tests use mocked pipeline functions (no real API calls)
  </behavior>

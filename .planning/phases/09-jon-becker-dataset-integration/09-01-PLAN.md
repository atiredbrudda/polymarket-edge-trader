**GREEN phase - Implement `src/datasources/jbecker.py`:**

Create `JBeckerDataset` class:
```python
class JBeckerDataset:
    """Query Jon Becker's Parquet dataset using DuckDB.

    Provides SQL interface to 33.5GB Parquet trade history.
    Uses parameterized queries ($1, $2) for security.
    Uses filter pushdown for performance (only matching rows loaded).
    """
```

Methods:
- `__init__(self, data_path: str)` - stores Path, derives trades_path as data_path/polymarket/trades
- `is_available(self) -> bool` - checks trades_path exists AND has trades_*.parquet files
- `query_trader_history(self, trader_address: str, limit: Optional[int] = None) -> list[dict]`:
  - CRITICAL: Use parameterized queries: `WHERE LOWER(maker) = LOWER($2) OR LOWER(taker) = LOWER($2)`
  - Pattern: `str(self.trades_path / "trades_*.parquet")`
  - Use `duckdb.execute(query, [pattern, trader_address])` -- NEVER f-strings
  - If limit: append `LIMIT {int(limit)}` (safe: limit is validated int)
  - Return `result.fetchdf().to_dict('records')` for small results
  - If `not self.is_available()`: raise FileNotFoundError with download URL
  - ORDER BY timestamp DESC
- `query_market_trades(self, asset_id: str, limit: Optional[int] = None) -> list[dict]`:
  - Query by makerAssetId or takerAssetId using parameterized query
  - Same pattern: parameterized, filter pushdown, ordered by timestamp DESC
- `get_trade_count(self, trader_address: str) -> int`:
  - COUNT query with parameterized address
- `get_date_range(self, trader_address: str) -> tuple[int, int] | None`:
  - MIN/MAX timestamp query with parameterized address
  - Returns (earliest_timestamp, latest_timestamp) or None if no trades
- `get_dataset_info(self) -> dict`:
  - Returns: file_count (number of parquet files), total_rows (approximate), date_range (min/max timestamp across all files)
  - Uses `SELECT COUNT(*) FROM read_parquet($1)` for total rows

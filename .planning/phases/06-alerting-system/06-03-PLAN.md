    1. Create src/alerts/delivery.py with:

       AlertDeduplicator class (in-memory TTL-based):
       - __init__(self, ttl_minutes: int = 60)
       - should_send(self, market_id: str, direction: str, event_type: str, computed_at: datetime) -> bool
       - Uses set of (market_id, direction, event_type, computed_at) tuples
       - Cleans expired entries on each should_send call

       AlertDeliveryResult frozen dataclass:
       - success: bool
       - market_id: str
       - direction: str
       - event_type: str
       - error: str | None = None

       deliver_signal_alerts function:
       ```python
       def deliver_signal_alerts(
           session: Session,
           alerter: TelegramAlerter,
           deduplicator: AlertDeduplicator | None = None,
           window_hours: int = 24,
       ) -> list[AlertDeliveryResult]
       ```

       Pipeline flow:
       a. Call get_ranked_signals(session, window_hours=window_hours, limit=100) from signals.pipeline
       b. For each SignalResult:
          - Call detect_signal_event(session, signal.market_id, signal.direction) from alerts.detector
          - If event_type is None, skip (no alert)
          - If deduplicator and not deduplicator.should_send(...), skip (duplicate)
          - Query Market table for market.question (session.query(Market).filter_by(condition_id=signal.market_id).first())
          - Call get_expert_position_details(session, signal.market_id, signal.expert_addresses) from alerts.formatter
          - Call format_signal_alert(event_type, market_question, signal, expert_positions) from alerts.formatter
          - Try: alerter.send(message), append success AlertDeliveryResult
          - Except Exception: log error, append failure AlertDeliveryResult, CONTINUE (don't block pipeline - user locked decision)
       c. Log summary: N/M alerts delivered successfully
       d. Return list of AlertDeliveryResult

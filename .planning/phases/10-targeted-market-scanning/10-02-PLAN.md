3. Add new method `ingest_targeted_markets`:
```python
def ingest_targeted_markets(
    self,
    niches: tuple[str, ...] = (),
    end_date_max: datetime | None = None,
) -> int:
    """Fetch and persist markets matching niche and time filters via Gamma API.

    Uses Gamma API for server-side filtering instead of fetching all markets.
    For each niche, queries Gamma API with tag filter. If end_date_max provided,
    includes it as additional filter. Results are unioned (OR semantics).

    Falls back to CLOB API + client-side filtering if Gamma client unavailable.

    Args:
        niches: Tuple of niche category strings (e.g., ("esports", "crypto"))
        end_date_max: Maximum end date for market closing time filter

    Returns:
        Count of markets processed
    """
```

Implementation logic for `ingest_targeted_markets`:
- If `self.gamma_client` is None, log warning and fall back to `ingest_active_markets()` with post-fetch client-side filtering
- For each niche in `niches`, call `self.gamma_client.get_markets(tag=niche, end_date_max=end_date_max, closed=False)`
- Union results across niches (deduplicate by condition_id)
- If no niches provided but end_date_max is, query Gamma without tag filter
- Convert Gamma API market dicts to MarketResponse objects (map fields: `condition_id`, `question`, `end_date_iso` from `end_date`, `active` from `closed` inverted, `category` from `tags` list)
- Persist using same upsert logic as `ingest_active_markets` (copy the existing market upsert loop)
- Log count of markets fetched per niche

4. Update `run_full_sweep` signature to accept filter parameters:
```python
def run_full_sweep(
    self,
    use_jbecker: bool = True,
    use_blockchain_fallback: bool = True,

    closing_within: str | None = None,
) -> dict:
```

In Step 1 of `run_full_sweep`, add branching logic:
```python
# Step 1: Ingest markets (targeted or full)
if niches or closing_within:
    # Parse closing_within to end_date_max
    end_date_max = None
    if closing_within:
        end_date_max = parse_closing_within(closing_within)
        logger.info(f"Filtering markets closing before {end_date_max}")

    overall_stats["markets_ingested"] = self.ingest_targeted_markets(
        niches=niches,
        end_date_max=end_date_max,
    )
else:
    # No filters - use existing full scan behavior
    overall_stats["markets_ingested"] = self.ingest_active_markets()
```

The rest of `run_full_sweep` (Steps 2 and 3 - trader discovery and backfill) remains unchanged.

    ```python
    class ExpertiseScore(Base):
        """Expertise score snapshot for a trader in a specific game.

        Stores score history for trend analysis and leaderboard generation.
        Each row is a point-in-time snapshot — new rows inserted on each scoring run.
        """
        __tablename__ = "expertise_scores"

        id: Mapped[int] = mapped_column(primary_key=True, autoincrement=True)
        trader_address: Mapped[str] = mapped_column(String(42), nullable=False)
        game_slug: Mapped[str] = mapped_column(String(100), nullable=False)
        raw_score: Mapped[Decimal] = mapped_column(Numeric(10, 6), nullable=False)
        percentile_rank: Mapped[Decimal | None] = mapped_column(Numeric(10, 6), nullable=True)
        win_rate_component: Mapped[Decimal] = mapped_column(Numeric(10, 6), nullable=False)
        concentration_component: Mapped[Decimal] = mapped_column(Numeric(10, 6), nullable=False)
        recency_component: Mapped[Decimal] = mapped_column(Numeric(10, 6), nullable=False)
        sample_size_component: Mapped[Decimal] = mapped_column(Numeric(10, 6), nullable=False)
        consistency_multiplier: Mapped[Decimal] = mapped_column(Numeric(10, 6), nullable=False)
        specialization_label: Mapped[str] = mapped_column(String(50), nullable=False)
        resolved_market_count: Mapped[int] = mapped_column(nullable=False)
        computed_at: Mapped[datetime] = mapped_column(default=datetime.utcnow, nullable=False)

        __table_args__ = (
            Index("ix_expertise_trader_game", "trader_address", "game_slug"),
            Index("ix_expertise_game_score", "game_slug", "raw_score"),
            Index("ix_expertise_computed_at", "computed_at"),
        )
    ```

    Add ExpertiseScore to imports in queries.py.

    **In src/pipeline/queries.py**, add these query functions:

    1. `get_game_leaderboard(session, game_slug, top_n=20, min_score=None) -> list[ExpertiseScore]`
       - Query latest ExpertiseScore entries per trader for a game
       - Use subquery to find max(computed_at) per trader per game, then join
       - Order by percentile_rank DESC (or raw_score DESC if percentile_rank is None)
       - Filter by min_score if provided (raw_score >= min_score)
       - Limit to top_n results
       - Returns list of ExpertiseScore ORM objects

    2. `get_trader_score_history(session, trader_address, game_slug=None, limit=50) -> list[ExpertiseScore]`
       - Query score history for a trader, optionally filtered by game
       - Order by computed_at DESC
       - Limit results (default 50)
       - Returns list of ExpertiseScore ORM objects

    3. `get_all_game_slugs_with_positions(session) -> list[str]`
       - Query distinct game slugs that have Position + MarketClassification data
       - Join Position -> MarketClassification -> TaxonomyNode
       - Filter for taxonomy nodes where node_type = 'game'
       - Returns list of game slug strings

    4. `get_positions_for_game(session, game_slug, trader_address=None) -> list[Position]`
       - Query positions in markets classified under a specific game slug
       - Join Position -> MarketClassification -> TaxonomyNode
       - Filter TaxonomyNode.slug LIKE '{game_slug}%' (captures game and sub-nodes)
       - Optionally filter by trader_address
       - Returns list of Position ORM objects

  <action>
    **Create src/pipeline/scoring_pipeline.py** — the orchestration layer that:

    1. Defines `LeaderboardEntry` frozen dataclass:
       - rank: int
       - trader_address: str
       - game_slug: str
       - raw_score: Decimal
       - percentile_rank: Decimal
       - win_rate: Decimal | None
       - realized_pnl: Decimal
       - trade_count: int
       - unique_markets: int
       - last_active: datetime | None
       - specialization_label: str

    2. `compute_game_scores(session, game_slug, weights=None, now=None) -> list[LeaderboardEntry]`
       - Get all traders with positions in this game via get_positions_for_game
       - Group positions by trader_address
       - For each trader:
         a. Filter to resolved, non-void positions; skip if < 5
         b. Calculate win rate via calculate_win_rate
         c. Calculate esports_concentration (volume in this game's positions / all positions volume)
           Use calculate_total_volume on position trades. For simplicity, use position.size * position.avg_entry_price as volume proxy since we have positions not raw trades
         d. Calculate game_concentration (same approach, game vs eSports)
         e. Get consistency_score and consistency_signal from the PerformanceSnapshot table (if exists) OR compute from positions using calculate_consistency
         f. Call calculate_expertise_score with pre-computed values
       - Collect all raw_scores, call normalize_scores_to_percentiles
       - Persist ExpertiseScore entries to database (INSERT new rows, don't overwrite history)
       - Build LeaderboardEntry list with rank (1-indexed by percentile_rank DESC)
       - Return sorted list

    3. `compute_all_game_scores(session, weights=None, now=None) -> dict[str, list[LeaderboardEntry]]`
       - Get all game slugs via get_all_game_slugs_with_positions
       - Call compute_game_scores for each game
       - Return dict mapping game_slug -> leaderboard

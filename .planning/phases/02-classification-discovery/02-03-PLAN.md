  **src/pipeline/classify.py - ClassificationPipeline:**

  Class with these methods:

  __init__(self, session_factory: sessionmaker, taxonomy_path: Path | None = None):
  - Loads taxonomy from taxonomy_path (or settings.taxonomy_path if None)
  - Creates PatternMatcher from loaded taxonomy
  - Stores session_factory for DB operations

  sync_taxonomy_to_db(self) -> int:
  - Walks the loaded TaxonomyConfig hierarchy
  - For each node (root, games, tournaments, teams), upserts a TaxonomyNode row
  - Uses slug for matching (e.g. "esports", "esports.cs2", "esports.cs2.iem-katowice")
  - Stores patterns as JSON string in patterns_json field
  - Returns count of nodes synced
  - This enables: load YAML -> sync to DB -> all taxonomy data is queryable via SQL

  classify_market(self, market: Market) -> MarketClassification:
  - Uses PatternMatcher.classify_with_review(market.question)
  - Uses detect_market_type(market.question)
  - Looks up taxonomy_node_id from TaxonomyNode table by matching node_path
  - Returns MarketClassification object (not yet persisted)

  classify_all_markets(self) -> dict:
  - Queries all markets that don't have a MarketClassification yet
  - Classifies each one using classify_market
  - Persists classifications in batch (commit every 100)
  - Returns stats: {"classified": N, "flagged": M, "match_type": X, "prop_type": Y}

  classify_new_markets(self) -> dict:
  - Same as classify_all_markets but only processes markets without existing classification
  - For incremental runs after initial classification

  **src/discovery/trader_discovery.py:**

  discover_esports_traders(session: Session, min_trades: int = 5, min_volume: Decimal = Decimal("500")) -> list[str]:
  - Queries trades joined with market_classifications to find traders active in eSports-classified markets
  - Groups by trader_address
  - Filters by HAVING count(distinct trade_id) >= min_trades AND sum(size * price) >= min_volume
  - CRITICAL: Uses distinct trade_id to avoid Polymarket double counting
  - Returns list of qualifying trader addresses

  compute_and_store_positions(session: Session, trader_address: str) -> list[Position]:
  - Gets all eSports-classified markets this trader has traded in (join trades -> market_classifications)
  - For each (trader, market) pair: get trades, call calculate_position from position_tracker
  - Upsert Position row: if exists, update all fields; if new, insert
  - Records entry_timestamp and first_trade_timestamp for entry timing analysis
  - Returns list of Position objects

  refresh_all_positions(session: Session, trader_addresses: list[str] | None = None) -> dict:
  - If trader_addresses is None, gets all discovered eSports traders
  - For each trader, calls compute_and_store_positions
  - Returns stats: {"traders_processed": N, "positions_computed": M, "positions_open": X, "positions_flat": Y}

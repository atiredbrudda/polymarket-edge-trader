## Goal

Create and run a migration script to update existing 1.6M trades with synthetic `graph_*` market_ids to real `condition_id` values from `token_catalog`.

## Scope

### In Scope
- Create migration script `scripts/migrate_graph_market_ids.py`
- Extract `asset_id` from synthetic market_id format: `graph_{txhash}_{asset_id}`
- Batch update trades in chunks to avoid DB lock issues
- Log progress and unknown tokens
- Report migration statistics

### Out of Scope

- Schema changes
- Position rebuild (separate command after migration)

## Implementation Details

### Market ID Format
```
graph_0x{transactionHash}_{asset_id}
Example: graph_0xabc123def456_123456789
```

### Migration Algorithm
```python
1. Query all trades WHERE market_id LIKE 'graph_%'
2. For each trade:
   a. Parse asset_id from market_id (split by '_', take last part)
   b. Look up asset_id in token_catalog → condition_id
   c. If found, queue for update
   d. If not found, log for manual review
3. Batch update in chunks of 10,000
4. Report: updated_count, not_found_count, total
```

### SQL Update Pattern
```sql
UPDATE trades SET market_id = :condition_id WHERE id = :trade_id
```

### Script Structure
```python
#!/usr/bin/env python3
"""Migrate Graph trades from synthetic to real market_ids."""

import re
from sqlalchemy import create_engine
from src.db.models import Trade, TokenCatalog

def extract_asset_id(market_id: str) -> str:
    """Extract asset_id from graph_{txhash}_{asset_id} format."""
    parts = market_id.split('_')
    return parts[-1] if len(parts) >= 3 else None

def migrate(session, batch_size=10000):
    # Load token_catalog into memory
    catalog = {row.token_id: row.condition_id for row in session.query(TokenCatalog).all()}
    
    # Query orphaned trades
    orphaned = session.query(Trade).filter(Trade.market_id.like('graph_%')).all()
    
    updated = 0
    not_found = 0
    batch = []
    
    for trade in orphaned:
        asset_id = extract_asset_id(trade.market_id)
        if asset_id and asset_id in catalog:
            trade.market_id = catalog[asset_id]
            updated += 1
        else:
            not_found += 1
            continue
        
        # Commit batch
        if len(batch) >= batch_size:
            session.commit()
            batch = []
    
    session.commit()
    return updated, not_found
```

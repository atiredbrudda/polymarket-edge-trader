<feature>
  <name>Performance Metrics Calculator</name>
  <files>src/evaluation/__init__.py, src/evaluation/metrics.py, tests/test_metrics.py</files>
  <behavior>
    Follow the pure-function, duck-typed pattern established by position_tracker.py.
    All functions accept simple data objects (dataclasses or duck-typed), not ORM models.
    All financial math uses Decimal.

    Functions to implement:

    1. calculate_realized_pnl(positions: list) -> Decimal
       - Input: list of position-like objects with fields: size, direction, avg_entry_price, resolved (bool), outcome (str|None), pnl (Decimal|None)
       - Filter: only positions where resolved=True AND outcome != "void"
       - Sum their pnl values
       - Return total realized PnL as Decimal
       - Cases:
         - Empty list -> Decimal("0")
         - All voided -> Decimal("0")
         - Mix of wins/losses -> net sum
         - Unresolved positions excluded

    2. calculate_win_rate(positions: list) -> dict
       - Input: same position-like objects
       - Filter: resolved=True AND outcome not in ("void", "flat", None)
       - Count wins (outcome == "win") and losses (outcome == "loss")
       - Return {"wins": int, "losses": int, "total": int, "win_rate": Decimal|None}
       - win_rate = wins / total * 100 (as Decimal percentage), None if total == 0
       - Cases:
         - No resolved -> {"wins": 0, "losses": 0, "total": 0, "win_rate": None}
         - All wins -> win_rate = Decimal("100")
         - 3 wins 2 losses -> win_rate = Decimal("60")

    3. calculate_total_volume(trades: list) -> Decimal
       - Input: list of trade-like objects with size and price attributes
       - Return sum of abs(size * price) for all trades
       - Cases:
         - Empty list -> Decimal("0")
         - Single trade size=10, price=0.65 -> Decimal("6.5")

    4. calculate_unrealized_pnl(position, current_price: Decimal) -> dict
       - Input: single position-like object + current market price
       - If position is FLAT (size == 0): return {"pnl": Decimal("0"), "unrealized": True}
       - LONG: pnl = size * (current_price - avg_entry_price)
       - SHORT: pnl = abs(size) * (avg_entry_price - current_price)
       - Return {"pnl": Decimal, "unrealized": True, "direction": str, "current_price": Decimal}
       - Cases:
         - LONG, entry=0.40, current=0.60, size=100 -> pnl = Decimal("20")
         - SHORT, entry=0.70, current=0.30, size=50 -> pnl = Decimal("20")
         - FLAT -> pnl = Decimal("0")

    5. aggregate_trader_metrics(positions: list, trades: list, unrealized_positions: list[tuple] = None) -> dict
       - Combines above functions into single trader metrics snapshot
       - unrealized_positions: list of (position, current_price) tuples for unresolved markets
       - Return:
         {
           "realized_pnl": Decimal,
           "unrealized_pnl": Decimal,
           "total_pnl": Decimal,  # realized + unrealized
           "win_rate": dict,  # from calculate_win_rate
           "total_volume": Decimal,
           "resolved_markets": int,
           "unresolved_markets": int,
         }
  </behavior>
